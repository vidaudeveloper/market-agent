# TTS 全案代运营报价逻辑 / Pricing Logic

> 来源：`d:\策划师agent\【全案代运营计费逻辑】.txt`  
> 结构化配置：`templates/config/pricing-logic.json`

## 何时使用

- 写 **TTS 合作方案**、**运营测算**、**增长全案** 中涉及 **报价 / Package / 季度费用**
- 用户问「代运营多少钱」「按 GMV 目标怎么报价」
- 与 `tts-partnership-proposal`、`tts-operation-model`、`tts-full-case` **联合执行**

## 推理链（必须按顺序）

```
GMV 目标
  → 倒推月视频量（自制 BGC）+ 达人视频量 + 是否加直播
  → 匹配岗位人手（店铺/编导/剪辑/达人BD）
  → 按摊销与按条双算法取中间值
  → 输出月费 + 季度费 + 三级套组对照
```

## 岗位与单价（USD/月）

| 岗位 | 规则 | 默认单价 |
|------|------|----------|
| 店铺运营 + 投流 | 每店铺 1 人摊销 | **1000** |
| 编导脚本 | 每店铺 1 人摊销 | **1000** |
| 剪辑 | ① 每 **150** 条视频 1 人 × **2200**/人；② **20** USD/条（取中间值） | 复杂品 20–60/条；服装 10–15/条 |
| 达人 BD | ① 每 **60** 条达人视频 1 人 × **1500**/人；② **30** USD/条（取中间值） | 见上 |

### 剪辑按条单价调整

| 品类特征 | 建议 USD/条 |
|----------|-------------|
| 形态复杂、AI 自制成本高（如电动车 Narrak） | 20–60 |
| 服装等 AI 生成容易 | 10–15 |
| 默认（未说明品类） | 20 |

## 计算公式

```text
剪辑月费 = avg( ceil(自制条数/150) × 2200 , 自制条数 × 单价 )
达人BD月费 = avg( ceil(达人条数/60) × 1500 , 达人条数 × 30 )
月运营费 = 1000 + 1000 + 剪辑月费 + 达人BD月费
季度费 = 月运营费 × 3
```

## 标准示例（Narrak 启动期前 3 月）

| 输入 | 数值 |
|------|------|
| 月达人素材 | 50 条 |
| 月自制视频 | 300 条 |

| 岗位 | 计算 | 月费 USD |
|------|------|----------|
| 店铺+投流 | 固定 | 1000 |
| 编导 | 固定 | 1000 |
| 剪辑 | avg(4400, 6000) | **5200** |
| 达人 BD | avg(1500, 1500) | **1650** |
| **合计** | | **8850/月** |
| **季度** | ×3 | **26550** |

## 写入报告的强制章节

在 `tts-partnership-proposal` 或 `tts-operation-model` 中须包含：

1. **GMV 目标 → 内容量倒推表**（达人条数 / 自制条数 / 直播小时）
2. **人手匹配表**（岗位、人数、摊销依据）
3. **报价明细表**（四岗位 + 合计，注明双算法取中）
4. **三级套组**（可基于月费 × 系数，或引用 `agency-defaults.json` 作档位标签）
5. **假设与边界**（不含广告费、达人佣金、样品费等须单列）

## 与出海匠数据的衔接

报价前的 **分析环节** 须先跑出海匠（见 `chuhaijiang-data`）：

| 出海匠数据 | 用于报价 |
|------------|----------|
| 竞品店铺 GMV/渠道结构 | 倒推合理 GMV 目标与达人占比 |
| 达人 GMV/标签 | Package 达人条数、BD 工作量 |
| 爆款商品与销量 | 自制视频条数、剪辑单价档位 |
| **截图证据** | 飞书报告「数据来源」章节佐证 |

## 输出检查

- [ ] 有 GMV → 视频量倒推，不是凭空写 Package 数字
- [ ] 四岗位费用均有计算过程（至少一行公式）
- [ ] 月费 + 季度费数字一致
- [ ] 品类影响剪辑单价时已说明
- [ ] 已引用出海匠抓取时间与截图/表格来源

## 禁止

- 只给「基础版/标准版/全案版」三档数字而无倒推过程
- 复制 `agency-defaults.json` 固定档位而不按 GMV 重算
- 写报价章节却未跑出海匠竞品/达人数据（须说明无数据并标注推断）
