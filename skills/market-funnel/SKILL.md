# 转化漏斗分析 / Sales Funnel Analysis

## 触发方式
- "分析 https://xxx.com 的转化漏斗"
- "帮我看下转化路径有哪些问题"
- "哪里流失用户最多"

## 第一步：识别漏斗类型

| 类型 | 典型步骤 | 核心指标 |
|------|----------|----------|
| 线索获取 | 落地页→表单→感谢页→培育→通话 | 线索到成交率 |
| SaaS 试用 | 首页→定价→注册→上手→升级 | 试用到付费率 |
| SaaS Demo | 首页→功能→Demo→通话→成交 | Demo 到成交率 |
| 电商 | 商品页→购物车→结算→追加销售→完成 | 购物车到购买率 |

## 第二步：绘制漏斗地图
用文本绘制访客从进入到转化的每一步：
```
访问来源 → 首页 → 定价页 → 注册 → 激活 → 付费
  100%       60%      25%     12%     8%      3%
```

## 第三步：逐页分析
每页评分五个维度（0-10）：清晰度、连贯性、动机、摩擦度、信任度

## 第四步：漏斗指标基准

| 类型 | 良好 | 优秀 | 顶级 |
|------|------|------|------|
| 线索获取(表单) | 3-5% | 5-10% | 10-20% |
| SaaS 免费试用 | 2-5% | 5-10% | 10-15% |
| 试用到付费 | 10-15% | 15-25% | 25-40% |
| 电商浏览到购买 | 1-3% | 3-5% | 5-8% |

## 第五步：优化建议
- 漏斗顶部：标题A/B测试、社交证明位置、页面速度
- 漏斗中部：案例研究、功能对比页、互动演示
- 漏斗底部：定价页重设计、结账摩擦减少、风险逆转
- 购后：上手邮件序列、追加销售、推荐计划
