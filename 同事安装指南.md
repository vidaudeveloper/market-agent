# 同事安装指南（Hermes Agent）

> 给**第一次使用**本项目的同事。全程约 10–15 分钟。

**仓库地址：** https://github.com/vidaudeveloper/market-agent

---

## 你要做什么（一句话）

1. 把仓库**克隆到你电脑**
2. 在 Hermes 里**打开这个文件夹**当工作区
3. 对 Agent 说：**「帮我安装 Vidau Market Agent」**（或复制下方提示词）
4. 按提示完成**飞书授权**、**出海匠登录**（要做数据抓取时）

---

## 方式一：让 Hermes 帮你装（推荐）

### 第 1 步：克隆仓库

在终端执行（路径可自定）：

```bash
git clone https://github.com/vidaudeveloper/market-agent.git
cd market-agent
```

Windows 也可：装 [Git](https://git-scm.com) 后，在资源管理器地址栏输入 `cmd`，再执行上面命令。

### 第 2 步：Hermes 打开文件夹

在 Hermes Agent 中：**Open Workspace / 打开工作区** → 选择刚克隆的 `market-agent` 文件夹。

### 第 3 步：复制下面这段话发给 Hermes

```
请帮我安装 Vidau Market Agent。

仓库我已克隆到本机，当前工作区就是 market-agent 根目录。
请阅读 AGENT.md、同事安装指南.md、ENV_SETUP.md，然后依次完成：

1. 运行 setup.bat（Windows）或 bash setup.sh（Mac）
2. 若没有 .env：从 .env.example 复制，并提示我向管理员索取飞书 FEISHU_APP_ID / FEISHU_APP_SECRET
3. 运行 install-hermes-skill.bat（或 bash scripts/install-hermes-skill.sh），然后提醒我 /reload-skills
4. 运行 install-mcp.bat（或 bash install-mcp.sh），并帮我配置 Hermes 的 vidau-market MCP，然后 /reload-mcp
5. 检查 node -v（需 18+）、node scripts/feishu-diagnose.js
6. 引导我运行 feishu-connect.bat 完成飞书 OAuth
7. 最后执行 /vidau-market-agent，问我第一个营销任务

注意：写报告和 QuickChart 出图不需要 AI_API_KEY；飞书导出禁止用 Hermes 内置飞书插件，必须用 scripts/feishu-export.js。
```

### 第 4 步：按 Agent 提示点几下

| 步骤 | 你要做的 |
|------|----------|
| 飞书 `.env` | 向**团队管理员**私聊索取 `.env` 或 App ID/Secret，粘贴到本机 `.env` |
| 飞书连接 | 双击 `feishu-connect.bat`，浏览器点「同意」 |
| 出海匠 | 要做 TikTok 数据时，对 Agent 说「帮我登录出海匠」，或运行 `node scripts/chuhaijiang-fetch.js screenshot --login` |
| 验证 | 聊天输入 `/`，应出现 **`vidau-market-agent`** |

## 方式二：自己手动装

| 顺序 | Windows | Mac/Linux |
|------|---------|-----------|
| 1 依赖 | 双击 `setup.bat` | `bash setup.sh` |
| 2 飞书配置 | 复制 `.env.example` → `.env`，填入管理员给的凭证 | 同上 |
| 3 斜杠技能 | `install-hermes-skill.bat` | `bash scripts/install-hermes-skill.sh` |
| 4 MCP | `install-mcp.bat`，再按屏幕提示执行 `hermes mcp add ...` | `bash install-mcp.sh` |
| 5 飞书 OAuth | `feishu-connect.bat` | `bash scripts/feishu-connect.sh` |
| 6 开始使用 | Hermes 输入 `/vidau-market-agent` | 同上 |

详细说明：[HERMES.md](HERMES.md) · [MCP-USER-GUIDE.md](MCP-USER-GUIDE.md) · [FEISHU-USER-GUIDE.md](FEISHU-USER-GUIDE.md)

## 装好后怎么用

在 Hermes 输入 `/vidau-market-agent`，然后直接说任务，例如：

- 「帮某品牌做泰国 TikTok 全案，查出海匠竞品，导出飞书」
- 「查 beauty 类目达人+店铺，出报告带图表」
- 「把 output 里某份报告导出到我的飞书」

Agent 会自动：读技能 → 拉出海匠数据 → 写 Markdown → 导出飞书。

---

## 常见问题

| 问题 | 处理 |
|------|------|
| 没有 `/vidau-market-agent` | 重跑 `install-hermes-skill`，聊天输入 `/reload-skills` |
| MCP 工具不可用 | 确认 `install-mcp` 已执行，Hermes 里 `/reload-mcp` |
| 飞书导出失败 | 检查 `.env` 凭证 + 重新 `feishu-connect.bat` |
| 出海匠无数据 | 浏览器登录出海匠：`screenshot --login` |
| 要 AI 生图 | 可选填 `AI_API_KEY`；**默认不需要** |

## 管理员分发清单（给你用的）

推送到 GitHub 前确认：

- [ ] `.env` **不在** Git 里（已在 `.gitignore`）
- [ ] 同事通过**私聊**领取飞书 `.env` 或凭证
- [ ] 仓库为 **Private**（推荐）
- [ ] 若 Secret 曾误提交过 Git，在飞书开放平台**轮换 Secret**

把本文档 + 仓库链接发给同事即可。
