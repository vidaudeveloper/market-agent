# 同事安装指南（Hermes Agent）

> 给**第一次使用**本项目的同事。全程约 10–15 分钟。

**仓库地址：** https://github.com/vidaudeveloper/market-agent

---

## 你要做什么（一句话）

1. 把仓库**克隆到你电脑**
2. 在 Hermes 里**打开这个文件夹**当工作区
3. 对 Agent 说：**「帮我安装这个工具」**（或复制下方完整提示词）
4. Agent 会自动装依赖、技能、两套 MCP，并**提示你填出海匠 API Key**
5. 按提示完成**飞书授权**（要做飞书导出时）

---

## 方式一：让 Hermes 帮你装（推荐）

### 第 1 步：克隆仓库

在终端执行（路径可自定）：

```bash
git clone https://github.com/vidaudeveloper/market-agent.git
cd market-agent
```

Windows 也可：装 [Git](https://git-scm.com) 后，在资源管理器地址栏输入 `cmd`，再执行上面命令。

### 第 2 步：Hermes 打开文件夹

在 Hermes Agent 中：**Open Workspace / 打开工作区** → 选择刚克隆的 `market-agent` 文件夹。

### 第 3 步：复制下面这段话发给 Hermes

```
请帮我安装 Vidau Market Agent（market-agent 仓库）。

仓库我已克隆到本机，当前工作区就是 market-agent 根目录。
请阅读 AGENT.md、同事安装指南.md、ENV_SETUP.md，然后依次完成：

【一键脚本 — 优先】
- Windows: 运行 install-all.bat
- Mac/Linux: bash install-all.sh
若脚本某步失败，再拆步手动执行下面各条。

【拆步安装（与 install-all 等价）】
1. setup.bat / bash setup.sh（Node 18+、npm、Playwright）
2. 若没有 .env：从 .env.example 复制，并提示我向管理员索取飞书 FEISHU_APP_ID / FEISHU_APP_SECRET
3. install-hermes-skill.bat / bash scripts/install-hermes-skill.sh → 提醒我 /reload-skills
4. install-mcp.bat / bash install-mcp.sh（vidau-market MCP，合并写入不覆盖已有配置）
5. node scripts/install-chuhaijiang-skillhub.js（SkillHub 安装出海匠官方 skill）
6. node scripts/install-chuhaijiang-open-mcp.js（出海匠开放平台 HTTP MCP）
7. node scripts/feishu-diagnose.js；引导我 feishu-connect.bat / bash scripts/feishu-connect.sh

【出海匠 API Key — 必须引导】
按已安装的 chuhaijiang skill 内 references/setup.md 执行：
- 问我：「你有出海匠 API Key 吗（sk_live_ 开头）？」
- 没有则引导：登录 chuhaijiang.com → developer.chuhaijiang.com 创建密钥
- 有 Key 则帮我运行：node scripts/install-chuhaijiang-open-mcp.js --api-key <Key>
- 提醒我 Hermes 输入 /reload-mcp
- 自行验证：调用出海匠 MCP 的 account_info（工具名可能带前缀如 mcp__chuhaijiang__account_info）
- 返回账户余额则成功；401 则 Key 错误；找不到工具则排查配置后重试

【两套出海匠能力说明（装完后告诉我）】
| 组件 | 作用 |
|------|------|
| vidau-market MCP（仓库内） | Playwright 抓取、飞书导出、auth_chuhaijiang_login 浏览器登录 |
| chuhaijiang 开放平台 MCP | search / get_detail 等实时 API（需 sk_live_ Key） |
| SkillHub chuhaijiang skill | 配 MCP、API Key、验证 account_info 的官方指引 |
| 仓库 skills/chuhaijiang-data | 全案 workflow、pipeline 脚本 |

【约束】
- 写报告和 QuickChart 出图不需要 AI_API_KEY
- 飞书导出禁止用 Hermes 内置飞书插件，必须用 scripts/feishu-export.js
- MCP 配置用 merge 方式，不要覆盖我已有的其他 MCP 服务

【收尾】
- /reload-skills 和 /reload-mcp
- 执行 /vidau-market-agent，问我第一个营销任务
- 简要汇报：vidau-market MCP、出海匠开放平台 MCP、account_info 验证结果
```

### 第 4 步：按 Agent 提示点几下

| 步骤 | 你要做的 |
|------|----------|
| 飞书 `.env` | 向**团队管理员**私聊索取 `.env` 或 App ID/Secret，粘贴到本机 `.env` |
| 飞书连接 | 双击 `feishu-connect.bat`（Mac: `bash scripts/feishu-connect.sh`），浏览器点「同意」 |
| 出海匠 API Key | Agent 会问你；没有就去 https://developer.chuhaijiang.com 创建 `sk_live_` 密钥，发给 Agent 或自己运行 `node scripts/install-chuhaijiang-open-mcp.js --api-key sk_live_xxx` |
| 出海匠浏览器登录 | 仅 Playwright 抓取时需要：对 Agent 说「帮我登录出海匠」，或 `node scripts/chuhaijiang-fetch.js screenshot --login` |
| 重载 | Hermes 聊天输入 `/reload-skills` 和 `/reload-mcp` |
| 验证 | 聊天输入 `/`，应出现 **`vidau-market-agent`**；Agent 应能调通 `account_info` |

## 方式二：自己手动装

| 顺序 | Windows | Mac/Linux |
|------|---------|-----------|
| 0 一键 | 双击 `install-all.bat` | `bash install-all.sh` |
| 1 依赖 | 双击 `setup.bat` | `bash setup.sh` |
| 2 飞书配置 | 复制 `.env.example` → `.env`，填入管理员给的凭证 | 同上 |
| 3 斜杠技能 | `install-hermes-skill.bat` | `bash scripts/install-hermes-skill.sh` |
| 4 vidau MCP | `install-mcp.bat` | `bash install-mcp.sh` |
| 5 出海匠 skill | `node scripts/install-chuhaijiang-skillhub.js` | 同上 |
| 6 出海匠 MCP | `node scripts/install-chuhaijiang-open-mcp.js --api-key sk_live_xxx` | 同上 |
| 7 飞书 OAuth | `feishu-connect.bat` | `bash scripts/feishu-connect.sh` |
| 8 重载 | Hermes: `/reload-skills` + `/reload-mcp` | 同上 |
| 9 开始使用 | Hermes 输入 `/vidau-market-agent` | 同上 |

详细说明：[HERMES.md](HERMES.md) · [MCP-USER-GUIDE.md](MCP-USER-GUIDE.md) · [FEISHU-USER-GUIDE.md](FEISHU-USER-GUIDE.md)

## 装好后怎么用

在 Hermes 输入 `/vidau-market-agent`，然后直接说任务，例如：

- 「帮某品牌做泰国 TikTok 全案，查出海匠竞品，导出飞书」
- 「查 beauty 类目达人+店铺，出报告带图表」
- 「把 output 里某份报告导出到我的飞书」

Agent 会自动：读技能 → 拉出海匠数据（API 或 Playwright）→ 写 Markdown → 导出飞书。

## 常见问题

| 问题 | 处理 |
|------|------|
| 没有 `/vidau-market-agent` | 重跑 `install-hermes-skill`，聊天输入 `/reload-skills` |
| vidau-market MCP 不可用 | 确认 `install-mcp` 已执行，Hermes 里 `/reload-mcp` |
| 出海匠 API 401 | Key 错误或过期；去 developer.chuhaijiang.com 重建，重跑 `install-chuhaijiang-open-mcp.js --api-key` |
| 找不到 account_info 工具 | `/reload-mcp`；检查 `~/.hermes/config.yaml` 里 `chuhaijiang` 段 |
| SkillHub 安装失败（Windows） | 需 Git Bash 或 WSL；或手动 `curl -fsSL https://skillhub-1388575217.cos.ap-guangzhou.myqcloud.com/install/install.sh \| bash -s -- --cli-only` 后重试 |
| 飞书导出失败 | 检查 `.env` 凭证 + 重新 `feishu-connect.bat` |
| Playwright 出海匠无数据 | 浏览器登录：`screenshot --login`（与 API Key 是两套能力） |
| 要 AI 生图 | 可选填 `AI_API_KEY`；**默认不需要** |

## 管理员分发清单（给你用的）

推送到 GitHub 前确认：

- [ ] `.env` **不在** Git 里（已在 `.gitignore`）
- [ ] 同事通过**私聊**领取飞书 `.env` 或凭证
- [ ] 仓库为 **Private**（推荐）
- [ ] 若 Secret 曾误提交过 Git，在飞书开放平台**轮换 Secret**
- [ ] 同事各自在 developer.chuhaijiang.com 申请**自己的** `sk_live_` API Key（或团队共用 Key 由你私聊分发）

把本文档 + 仓库链接发给同事即可。
